# Spyder
Spyder Directory Web


Installation:

cd spyder
pip3 install colorama



Usage:
-d', '--domain'
-w', '--wordlist'
-c', '--cookie'
-u', '--ua', action="store", help="User agent to set"
-o', '--outfile', action="store", help="Output file", default="results.txt"

example:
python3 spyder.py -d http://testphp.vulnweb.com -w 123.txt
